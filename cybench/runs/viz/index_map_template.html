Explore walk-forward results by country, and cross-country summaries from the CY-Bench benchmark.

CY-Bench is a dataset and benchmark for subnational crop yield forecasting (maize and wheat), comparing classical, process-based, and machine-learning models. This site hosts interactive results: start with a country on the map, or open a global insight page below. More about the project on agml.org and the GitHub repository.
